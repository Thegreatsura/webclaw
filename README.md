# WebClaw

![Cover](https://raw.githubusercontent.com/ibelick/webclaw/main/apps/webclaw/public/cover.jpg)

Fast web client for OpenClaw.

[webclaw.dev](https://webclaw.dev)

Currently in beta.

## Installation

```bash
npx webclaw
```

The CLI will ask for a project name, env keys, and a port, then create the folder and start WebClaw.

WebClaw env vars: `CLAWDBOT_GATEWAY_URL` + `CLAWDBOT_GATEWAY_TOKEN` (or `CLAWDBOT_GATEWAY_PASSWORD`) â€” OpenClaw gateway/auth docs: https://docs.openclaw.ai/gateway (env precedence: https://docs.openclaw.ai/help/environment).
